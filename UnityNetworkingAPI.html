Unity Networking API

The Unity Networking API is a high level networking solution for Unity I wrote together with another programmer. We are currently using the API to create a cross-platform multiplayer game which requires a low latency. These are some of the features:

  • Sending messages over UDP connection.
  • Sending both Reliable and Unreliable functions
  • Syncing of GameObjects
